# 시스템 목표 및 로드맵

## 시스템 목표

Binance 크립토 + 한국투자증권 미국주식 자동 스윙 트레이딩 시스템.
Mac Mini M4에서 24/7 운용, 룰 기반 시그널 + 정량 검증 파이프라인.

## 자본 제약 분석 ($1-5K)

소규모 자본에서는 거래 비용이 수익률을 지배한다.

| 항목 | Binance Futures | 한투 미국주식 |
|------|----------------|--------------|
| 수수료 (편도) | maker 0.02% / taker 0.05% | 0.25% |
| 왕복 비용 | 0.04-0.10% | 0.50% |
| $3K 1회 왕복 비용 | $1.2-3.0 | $15 |
| 월 20회 기준 | $24-60 | $300 |

결론: 크립토 Futures를 우선 운용. 미국주식은 자본 $5K 이상, 월 10-15회 이하로 제한.

## Agent-vs-Agent 경쟁 테제

AI 트레이딩 에이전트가 시장에 늘어나면서, 에이전트 간 경쟁/협력 역학이 시장 미시구조에 영향을 미친다는 연구가 증가하고 있다.

주요 레퍼런스:
- **Alpha Arena (2025)**: LLM 6개 에이전트 실거래 경쟁. 4개가 손실 기록.
- **Wharton NBER (2025)**: RL 에이전트들이 명시적 통신 없이 자율적 담합 형성.
- **JaxMARL-HFT (arXiv 2511.02136)**: GPU 가속 Multi-Agent RL로 HFT 시뮬레이션.

시사점: 단일 에이전트 최적화만으로는 불충분. 다른 에이전트의 전략 디케이와 군집 행동을 고려한 설계 필요.

## Go/No-Go 기준표

페이퍼 트레이딩 3-6개월 후 아래 기준으로 실전 전환 판단.

| 지표 | GO | NO-GO |
|------|-----|--------|
| 페이퍼 Sharpe (3개월) | > 0.8 | < 0.5 |
| 승률 | > 45% | < 40% |
| 최대 낙폭 (MDD) | < 15% | > 25% |
| 크론 가동률 | > 99% | < 95% |
| vs Buy-and-Hold | 초과수익 양(+) | 음(-) 2개월 연속 |

회색 지대 (GO/NO-GO 사이): 1개월 추가 관찰 후 재판정.

## 거래 빈도 목표

비용 필터가 자연스럽게 거래 빈도를 조절한다.

- **크립토**: 월 15-30회 (Futures maker 0.02%로 빈도 허용)
- **미국주식**: 월 10-15회 (0.25% 수수료 → 높은 기대수익 시그널만 통과)

필터 기준: 기대수익 > 왕복비용 x 3 (안전 마진).
